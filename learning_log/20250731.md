# 반복문

반복문(Loop Statements)은 특정 조건이 만족될 때까지 코드를 반복 실행하는 제어문입니다.
## while 문
### 개념

`while` 문은 주어진 조건이 `true`인 동안 반복 실행됩니다.

```java
int i = 0;
while (i < 5) {
System.out.println("i의 값: " + i);
i++;
}
```

**설명**: `i`가 5보다 작을 때까지 증가하면서 반복됩니다.

#### 1~100 덧셈 예제

```java
int sum = 0;
int i = 0;

int sumOdd = 0;
int sumEven = 0;
int sumTotal = 0;
i = 0;

while (i < 101) {
    if( i % 2 == 1 ) {  // 홀수의 합
        sumOdd += i;
} else {                // 짝수의 합
sumEven += i;
}

    sumTotal += i;      // 전체의 합

    i++;
}

System.out.println(sumOdd);
System.out.println(sumEven);
System.out.println(sumTotal);
}
```
**결과**
```java
2500
2550
5050
```
**설명**: 1~100까지의 숫자 중 `홀수의 합`, `짝수의 합`, `전체의 합`을 구해 보여줍니다.

#### 구구단 예제

```java

int dan2 = 2;


while (dan2 < 10) {
    System.out.print(dan2 + "단\t\t\t");

}
System.out.println();

while (num < 10) {
    int dan = 2;



    while (dan < 10) {

        System.out.print(dan + " X " + num + " = " + (dan * num) + "\t");

}
```

## for 문

### 개념

for 문은 반복 횟수가 정해져 있을 때 사용하기 적합합니다.

```java
for (int i = 0; i < 5; i++) {
System.out.println("i의 값: " + i);
}
```

**설명**: `i`가 0부터 4까지 반복되면서 실행됩니다.

#### 구구단 예제

```java
// 구구단 출력하기
for (int i = 2; i < 10; i++) {
    System.out.print(i + "단\t\t\t");
}

System.out.println();

for (int i = 1; i < 10; i++) {
    for (int j = 2; j < 10; j++) {
        System.out.print(j + " X " + i + " = " + (j*i) + "\t");
    }
    System.out.println();
}
```

# for문 활용
## (1) *찍기

### ①별을 하나씩 증가시키기

```java
for (int i = 0; i < 6; i++) {
    for (int j = 0; j < i; j++) {
        System.out.print("*");
    }
    System.out.println();
    /*
         별 찍기 실행 예
         *          i = 0, j = 0
         **         i = 1, j = 0~1
         ***        i = 2, j = 0~2
         ****       i = 3, j = 0~3
         *****      i = 4, j = 0~4
     */
}
System.out.println();
```

### ③ Scanner 활용

```java

int line;
System.out.print("몇 줄의 별을 생성하시겠습니까? >>> ");
line = scan.nextInt();

for (int i = 0; i < line + 1; i++) {    // 오름차순
    for (int j = 0; j < i; j++) {
        System.out.print("*");
    }
    System.out.println();
}
System.out.println();

for (int i = 0; i < line; i++) {        // 내림차순
    for (int j = line; j > i; j--){
        System.out.print("*");
    }
    System.out.println();
}
System.out.println();

for (int i = line; i > 0; i--) {        // 내림차순
    for (int j = 0; j < i; j++) {
        System.out.print("*");
    }
    System.out.println();
}
```
