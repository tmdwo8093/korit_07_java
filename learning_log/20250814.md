# JSON

## 정의
JSON(JavaScript Object Notation) - 본래 자바스크립트 언어로부터 파생되어
자바스크립트 구문을 따르지만, 언어 독립형 데이터 포맷

프로그래밍 언어나 플랫폼에 독립적이기 때문에 구문 분석 및 JSON 데이터 생성을
위한 코드는 Java / Python등 다양한 언어에서 쉽게 이용 가능

